# AUTOWESTA---An-Automatic-Weather-Station-using-IoT
AutoWesta is an AI-powered IoT weather station that monitors temperature, humidity, pressure, rainfall, wind, soil moisture, air quality, and PM2.5. Built using Arduino UNO R4 Minima and ESP32-S3, it combines real-time sensing, cloud connectivity, SD card logging, and machine learning for accurate hyperlocal weather forecasting.
